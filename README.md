# 📊 Data Analytics Internship Portfolio

This repository contains all the tasks completed as part of my Data Analytics Internship. The project demonstrates a complete data analysis workflow, from data cleaning to data storytelling.

---

## 🎯 Project Objective

The objective of this project is to analyze sales data and extract meaningful insights that support data-driven business decisions.

---

## 🛠️ Tools & Technologies Used

✨ Python  
✨ Pandas  
✨ Matplotlib  
✨ Google Colab

## 📁 Tasks Overview

### 🔹 Task 1: Data Cleaning
✔ Handled missing values  
✔ Removed duplicate records  
✔ Converted date columns into proper format  
✔ Created new features (Year column)  

---

### 🔹 Task 2: Exploratory Data Analysis (EDA)
📊 Analyzed sales distribution  
📊 Performed category-wise analysis  
📊 Identified patterns and trends  
📊 Visualized insights using graphs

### 🔹 Task 3: Dashboard Creation
📈 Defined key performance indicators (KPIs)  
📈 Total Sales calculation  
📈 Category-wise performance visualization  
📈 Year-wise trend analysis  

---

### 🔹 Task 4: Data Storytelling & Analysis
🧠 Created a business narrative using insights  
🧠 Identified trends and growth patterns

## 📊 Key Insights

🚀 Sales showed an overall increasing trend

## 🧠 Conclusion

The analysis indicates a positive business growth trend. Data-driven insights help in making better strategic decisions and improving overall performance.
